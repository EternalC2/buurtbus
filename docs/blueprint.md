# **App Name**: Buurtbus Connect

## Core Features:

- User Authentication: Secure login and registration using email and password.
- Call-to-Bus: A prominent button for users to request the buurtbus, automatically sending their GPS location to the driver.
- Driver Notification: Sends notifications to available drivers that includes pickup location details
- AI Reistijd Estimation Tool: Uses a generative AI tool to provide users with an estimated arrival time of the bus, adjusted by historical data.
- Simple Navigation: Easy access to main sections (Home, History, Settings) through a bottom navigation bar.
- Admin Panel: Web-based admin panel to manage driver accounts, view active trips, and send notifications to drivers.
- Accessibility Features: Incorporate large buttons, clear icons, and readable typography for elderly users.

## Style Guidelines:

- Primary color: Soft light blue (#ADD8E6) to convey calmness and trust.
- Background color: Very light gray (#F5F5F5) to create a clean and open feel.
- Accent color: A muted green (#90EE90) to highlight important actions and provide a friendly touch. This choice is intended to be slightly left of blue on the color wheel.
- Body and headline font: 'PT Sans', a modern humanist sans-serif that provides both approachability and readability.
- Use simple, recognizable icons with rounded edges to match the friendly design. Design the 'Call-to-Bus' icon so that it strongly stands out.
- Maintain a clean and minimalist layout, ensuring the 'Call-to-Bus' button is always prominently displayed.
- Incorporate subtle animations, like the pulsing of an icon, to confirm user interactions and guide users.
